AWS Settings

Please follow these steps to set up AWS. Once setup, Read Aloud will use your AWS account to provide the Amazon Joanna and Matthew voices.

  1. Sign in to the AWS IAM console: https://console.aws.amazon.com/iam/home?#/users
  2. Create a user with "Programmatic Access".
  3. When asked to select permissions, choose "Attach existing policies directly", then search and add a policy called "AmazonPollyReadOnlyAccess".
  4. After creating the user, remember to download the access key, which includes an AccessKeyID and a SecretAccessKey. Enter them in the form below.
